Decision with Intelligence (Human and Artificial)

User Research  +   Ideation Workshops  +   Prototype  +  User Testing

This project aimed to create an intelligent data ecosystem capable of feeding internal and external users/clients with recommendations & insights related to Salesforce products

  • Salesforce (Internal)
  • Design Thinking and UX
  • 2018/2019

Project Objectives

01.

Understand the Customer Context.

Beyond metrics, the final product offered a context for the KPIs where users are able to understand the semantic of the information

02.

Use of AI and Automation

Power up interpretation of data from various sources systems allowing automation of actions via artificial intelligence.

03.

Use feedback from users to continuously learn

Using the feedback loop, users behaviour and action are sent back to the algorithms - Machine Learning - improving the precision of recommendations and insights.

Design Process:

During this project design played an important role from user research - interviews, observations and surveys - to user testing and handoff for final delivery using Agile as framework. Techniques applied: User Journey Mapping, User Story Mapping, Hypothesis Canvas, Surveys, User Interviews and Testing & High Fidelity Prototypes.

01
Discovery

Stakeholder Interviews, User Interviews and Observations, Quantitative Surveys, Data Gathering. Tools: SurveyMonkey, Google Hangouts

02
Define

Workshops, Problem Statement, Personas, Journey Mappings. Tools: Post-Its, Miro, Spreadsheet, Otter.ai

03
Develop

Low and High Fidelity Prototypes, User Testing Sessions, Iterations. Tools: SketchApp, InVision, Google Hangouts, Otter.ai

04
Deliver

Usability Test Report, User Story Mapping, Design Specifications. Tools: Tableau, InVision, Miro, Spreadsheets.

The result of this project produced our the most popular internal dashboard with thousands of access every week and more importantly: it helps to solve many customer issues everyday with a great experience.
